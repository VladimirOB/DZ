1. Пользователь вводит имя папки, программа сканирует папку, находит в ней текстовые файлы и формирует результирующий файл, содержащий столбец со словами без повторений

2. Создать программу-переводчик, имеющую следующее меню:
- добавить словарную пару
- удалить пару
- просмотреть все пары
- перевести слово в обоих направлениях
- перевести предложение (разбив на слова)
- сохранение / загрузка словаря

3. Пользователь вводит путь к папке с текстовыми файлами. Программа создаёт файл отчёта, который содержит глаголы с предлогами, с которыми эти глаголы употребляются

Пример:
бежать ( в, на, по, над, под, в направлении )
читать ( с, на )
