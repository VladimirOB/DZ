1. Добавить в программу с гаражем меню с указалями на функции-обработчики (без авторизации)

2. Добавить меню на указателей на функции в программу, которая обеспечивает различные уровни доступа при работе с гаражем:
- вход в программу по логину и паролю

- доступ для обычных пользователей при помощи меню:
	- добавить машину
	- просмотреть все машины
	- удаление только своих машин
	- выход

- доступ для админов при помощи меню:
	- добавить машину
	- просмотреть все машины
	- удаление любых машин
	- просмотр пользователей
	- добавление пользователя
	- удаление пользователя (с машинами)
	- выход
	
- сохранение / загрузка всех данных в программе

- реализция в классах

- использование полиморфизма, где это необходимо

3. Реализовать класс PriorityQueue, который позволяет хранить значения и извлекать их из очереди в соответствии с их приоритетом
Класс имеет следующие члены класса:
- int* buffer
- size_t MaxSize;

Методы:
- констр
- дестр
- EnQueue() - поместить в очередь
- DeQueue() - вытащить из очереди
- Show() - показать элемент с краю
- Print() - показать всю очередь
- Clear() - очистить всю очередь

Доп. задание: использовать для очереди компаратор (передать в конструктор)
