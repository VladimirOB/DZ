1. Пользователь вводит имя текстового файла, программа удаляет из него все слова "hello". Результат записывается в отдельный файл.
Первый файл никуда не девается.

2. Пользователь вводит имя текстового файла, программа делит файл на слова и сохраняет их в результирующий файл в столбик, отсортированными в алфавитном
порядке. Слова в файле разделяются пробемами, табуляцией или переходом на слудующую строку.

3. Пользователь вводит имена двух файлов, программа записывает в столбик слова в третий файл, только в том случае,
если слово встречается и в первом и во втором файле.

Использовать классы STL string и vector
